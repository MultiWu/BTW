# Security Policy

## Supported Versions

| Version          | Supported          |
| ---------------- | ------------------ |
| 0.0.1-beta.7     | :white_check_mark: |
| < 0.0.1-beta.7   | :x:                |

## Reporting a Vulnerability

If you found a vulnerability, please go to issues section and create new issue.
Choose label: "vulnerability" and describe what you found.
